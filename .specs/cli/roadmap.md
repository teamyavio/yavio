# CLI Roadmap

## Phase 7: CLI & Docker Packaging

- `yavio init` — interactive setup wizard
- `yavio up` / `yavio down` — docker-compose wrappers
- `yavio status` — platform health check
- `yavio doctor` — diagnostics
- Dockerfiles for dashboard and ingestion API
- Docker Compose production overrides
- Image publishing to Docker Hub / GitHub Container Registry
- **Milestone:** `yavio up` starts full platform from published Docker images
